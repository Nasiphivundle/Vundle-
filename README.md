# QA System in Rust using Burn

This project is a Transformer-based Question-Answering system in Rust. It reads `.docx` documents and answers questions about their content.

## Usage

- Place Word documents in `data/`
- Run training: `cargo run -- train`
- Run inference: `cargo run -- infer`

## Project Structure

- `src/` - Rust source code
- `docs/` - Project report
- `data/` - Word documents
